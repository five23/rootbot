# rootbot

Equation solving Slack bot.

1. `npm install`
2. `export SLACK_TOKEN=<your_token>`
2. `npm start`

With Docker

1. `docker build -t rootbot .`
2. `docker run --rm -it -e SLACK_TOKEN=<your_token> rootbot`
